# 📄 Desafio 013 - Módulo 03 - Capítulo 21

## 📌 Enunciado

Desafio proposto no capítulo **"Tabelas"** do curso de HTML e CSS da plataforma **Curso em Vídeo**.

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/fe6653ecdafa5b7275a5ecb4c36a8ca14f5043e7/aulas-pdf/21%20-%20Tabelas.pdf)

## 💡 Objetivo

Reproduzir tabelas com células mescladas, utilizando recursos de HTML para unir linhas e colunas, praticando organização estrutural e raciocínio lógico na montagem tabular.

## 🌐 Visualização

👉 [Parte 01 - tabela001.html](https://giulia-mb.github.io/html-semantico/modulo-03-html5-css3/desafios/d013/tabela001.html)  
👉 [Parte 02 - tabela002.html](https://giulia-mb.github.io/html-semantico/modulo-03-html5-css3/desafios/d013/tabela002.html)

## 🛠️ O que foi praticado

* Criação de tabelas com `table`, `tr` e `td`  
* Mesclagem horizontal com `colspan`  
* Mesclagem vertical com `rowspan`  
* Organização visual de células expandidas  
* Planejamento estrutural de linhas e colunas  
* Estilização básica com bordas e alinhamento centralizado  

## 📚 Aprendizados

* Como construir tabelas mais complexas em HTML  
* Diferença entre unir colunas e unir linhas  
* Importância de planejar a estrutura antes de escrever o código  
* Como células mescladas afetam o restante da tabela  
* Organização visual de dados em formato tabular  
* Uso de CSS para melhorar leitura e aparência  

## 📄 Estrutura do desafio

### `tabela001.html`

Tabela numérica contendo:

* células simples  
* colunas unificadas  
* linhas unificadas  
* distribuição numérica organizada  

### `tabela002.html`

Tabela alfabética contendo:

* células mescladas horizontais  
* células mescladas verticais  
* composição estrutural mais avançada  

## 🚀 Resultado

O desafio reforçou o uso de tabelas em HTML e mostrou como mesclar células corretamente para criar estruturas mais elaboradas.

---

## 💬 Observação

Este desafio exige atenção à lógica de posicionamento das células. É excelente para desenvolver percepção estrutural e domínio de `rowspan` e `colspan`, recursos muito importantes em tabelas reais.
