## 🛠️ O que foi praticado

* Uso de `background-image` com imagens locais e externas  
* Aplicação de fundo em elementos e no `body`  
* Uso de `background-repeat` para controlar repetição  
* Uso de `background-position` para posicionar imagens  
* Uso de `background-size` para redimensionar fundos  
* Combinação de cor sólida, gradiente e imagem como preenchimento  
* Criação de blocos com imagens de fundo personalizadas  
* Centralização de imagens no fundo da página  

## 📚 Aprendizados

* O fundo de um elemento não precisa ser apenas cor, também pode usar imagens  
* Por padrão, imagens pequenas se repetem automaticamente no eixo X e Y  
* `background-repeat: no-repeat;` impede repetição  
* `background-repeat: repeat-x;` repete apenas horizontalmente  
* `background-repeat: repeat-y;` repete apenas verticalmente  
* `background-position` permite escolher pontos como:
  - `left top`
  - `center center`
  - `right bottom`
* `background-size` pode usar:
  - `auto`
  - medidas em `px`
  - `%`
  - `cover`
  - `contain`
* Imagens externas precisam usar link direto do arquivo, não página do GitHub  
* Fundos bem configurados deixam o site mais bonito e profissional
