## 🛠️ O que foi praticado

* Uso de `background-image` com imagens locais e externas  
* Aplicação de fundo em elementos e no `body`  
* Uso de `background-repeat` para controlar repetição  
* Uso de `background-position` para posicionar imagens  
* Uso de `background-size` para redimensionar fundos  
* Uso de `background-attachment` para fundo fixo  
* Uso da shorthand `background`  
* Centralização vertical e horizontal com `position` + `transform`  
* Combinação de cor sólida, gradiente e imagem como preenchimento  

## 📚 Aprendizados

* O fundo de um elemento não precisa ser apenas cor, também pode usar imagens  
* `background-repeat: no-repeat;` impede repetição  
* `background-repeat: repeat-x;` repete horizontalmente  
* `background-repeat: repeat-y;` repete verticalmente  
* `background-position` permite posicionar a imagem em vários pontos da tela  
* `background-size: cover;` cobre toda a área disponível  
* `background-size: contain;` mostra toda a imagem sem cortar  
* `background-attachment: fixed;` mantém o fundo parado ao rolar a página  
* A shorthand `background` resume várias propriedades em uma linha  
* Elementos podem ser centralizados perfeitamente com:
  - `position: absolute`
  - `left: 50%`
  - `top: 50%`
  - `transform: translate(-50%, -50%)`
