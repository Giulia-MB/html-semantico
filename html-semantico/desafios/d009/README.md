# 📄 Desafio 009 - Módulo 01 - Capítulo 11

## 📌 Enunciado

Desafio proposto no material **"Seus vídeos favoritos"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-01/d009/desafio-videos.pdf)

## 💡 Objetivo

Criar um pequeno site com uma página principal contendo miniaturas de vídeos e páginas individuais para cada vídeo incorporado.

## 🛠️ O que foi praticado

* Organização de múltiplas páginas HTML  
* Uso de links internos entre páginas  
* Inserção de miniaturas clicáveis  
* Incorporação de vídeos do YouTube com `<iframe>`  
* Uso de links externos para playlists  
* Botão de retorno com imagem  

## 📚 Aprendizados

* A tag `<iframe>` permite incorporar vídeos externos sem hospedar o arquivo localmente  
* Links com `target="_blank"` abrem conteúdos externos em nova aba  
* A navegação entre páginas melhora a experiência do usuário  
* O uso de miniaturas torna a interface mais intuitiva  
* Esse desafio reforça conceitos de links, imagens e navegação entre páginas :contentReference[oaicite:0]{index=0}  

## 📄 Estrutura do desafio

O desafio contém:

* página inicial com quatro thumbs clicáveis  
* páginas individuais para cada vídeo  
* vídeo incorporado do YouTube  
* link para playlist completa  
* botão de voltar para a página principal  

## 🚀 Resultado

Mini site com galeria de vídeos favoritos, navegação intuitiva e integração com vídeos externos.

---

## 💬 Observação

Conforme orientado no desafio, os vídeos foram incorporados via YouTube em vez de hospedados localmente, o que evita arquivos pesados no projeto e melhora a compatibilidade entre navegadores.
