# 📄 Desafio 003 - Módulo 01 - Capítulo 06

## 📌 Enunciado

Desafio proposto no material **"Desafio do Mapa Mundi"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-01/d003/desafio-mapa-mundi.pdf)

## 💡 Objetivo

Montar um mapa mundi utilizando nove imagens diferentes, organizando corretamente sua posição na página.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/desafios/d003/)

## 🛠️ O que foi praticado

* Uso da tag `<img>` para inserir múltiplas imagens  
* Organização visual em linhas utilizando `<br>`  
* Uso de parágrafos para separação de conteúdo  
* Montagem de layout manual (estilo quebra-cabeça)  

## 📚 Aprendizados

* Diferença entre parágrafos e quebras de linha  
* Importância da organização visual sem uso de CSS  
* Como trabalhar com múltiplas imagens em sequência  
* Desenvolvimento de raciocínio lógico para posicionamento de elementos  

## 🚀 Resultado

Página HTML com Mapa mundi montado corretamente a partir da organização de nove imagens distintas.
