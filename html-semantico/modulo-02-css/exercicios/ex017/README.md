# 📄 Exercício 017 - Módulo 02 - Capítulo 14

## 📌 Enunciado

Exercício proposto no capítulo **"Vamos falar das Fontes"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/14%20-%20Fontes.pdf)

## 💡 Objetivo

Praticar a configuração de fontes em CSS, entendendo famílias tipográficas, tamanhos, pesos, estilos e a importação de fontes externas.

## 🌐 Visualização

👉 [fonte01.html - Fontes em CSS](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/modulo-02-css/exercicios/ex017/fonte01.html)

## 🛠️ O que foi praticado

### 🔤 Fontes em CSS

* Importação da fonte **Work Sans** pelo Google Fonts com `@import`  
* Definição de família tipográfica com fallback  
* Uso de `font-size` com unidade relativa `em`  
* Estudo de medidas absolutas e relativas em CSS  
* Aplicação de peso (`font-weight`) e estilo (`font-style`)  
* Uso da shorthand `font` para simplificar declarações  

## 📚 Aprendizados

### 🔤 Tipografia e estilização

* O CSS permite personalizar totalmente a aparência do texto  
* Fontes externas deixam o site mais bonito e moderno  
* É importante sempre definir fontes de fallback para garantir compatibilidade  
* Unidades relativas como `em` ajudam na responsividade  
* `font-weight` controla a espessura da fonte  
* `font-style` altera inclinação  
* A shorthand `font` resume várias propriedades em uma só linha  

## 📄 Estrutura do exercício

O exercício apresenta:

* título principal estilizado com fonte externa  
* subtítulo com shorthand font  
* parágrafos de exemplo para visualizar legibilidade  
* comentários explicativos sobre medidas e propriedades CSS

Página HTML demonstrando como configurar fontes externas e estilizar textos com CSS de forma mais profissional.
