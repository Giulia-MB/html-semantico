# 📄 Exercício 020 - Módulo 02 - Capítulo 15

## 📌 Enunciado

Exercício proposto no capítulo **"Seletores personalizados em CSS"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/15%20-%20Seletores%20personalizados.pdf)

## 💡 Objetivo

Praticar pseudo-classes e pseudo-elementos em CSS, aprendendo a criar efeitos visuais com interações do usuário, como hover, estados de links e conteúdo dinâmico.

## 🌐 Visualização

👉 [pseudoclasse.html - Pseudo-classes básicas](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/modulo-02-css/exercicios/ex020/pseudoclasse.html)  
👉 [hover.html - Efeito hover com texto oculto](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/modulo-02-css/exercicios/ex020/hover.html)  
👉 [links.html - Personalização de links](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/modulo-02-css/exercicios/ex020/links.html)

## 🛠️ O que foi praticado

### 🎯 pseudoclasse.html — Hover básico

* Uso da pseudo-classe `:hover`  
* Alteração de cor ao passar o mouse sobre elementos  
* Uso de `display: inline-block` para criar blocos interativos  

### 👀 hover.html — Texto oculto com hover

* Uso do seletor filho direto `>`  
* Ocultação de elementos com `display: none`  
* Exibição de conteúdo oculto ao passar o mouse  
* Estilização do texto revelado com fundo colorido  

### 🔗 links.html — Personalização de links

* Uso das pseudo-classes:
  * `:visited`
  * `:hover`
  * `:active`
* Remoção de sublinhado padrão  
* Mudança de cor conforme estado do link  
* Aplicação de classe especial em link específico  
* Uso de pseudo-elementos:
  * `::before`
  * `::after`

## 📚 Aprendizados

### 🎯 Pseudo-classes

* Pseudo-classes permitem estilizar elementos em estados específicos  
* `:hover` cria efeitos de interação visual  
* `:visited` altera aparência de links visitados  
* `:active` responde ao clique do usuário  

### ✨ Pseudo-elementos

* Permitem adicionar conteúdo visual sem alterar o HTML  
* `::before` insere conteúdo antes do elemento  
* `::after` insere conteúdo depois do elemento  

### 🧠 Seletores avançados

* O seletor `>` seleciona apenas filhos diretos  
* Combinar pseudo-classes com classes torna o CSS mais poderoso
