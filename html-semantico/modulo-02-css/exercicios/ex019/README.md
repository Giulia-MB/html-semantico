## 🛠️ O que foi praticado

### 🎯 Seletores em CSS

* Uso de seletor de elemento (`h1`, `h2`, `body`)  
* Uso de seletor por `id` com `#`  
* Uso de seletor por `class` com `.`  
* Aplicação de múltiplas classes em um mesmo elemento  
* Uso de `span` para estilizar trechos específicos do texto  
* Separação entre HTML e CSS com arquivo externo  

## 📚 Aprendizados

### 🎯 IDs e classes

* `id` serve para identificar elementos únicos  
* `class` permite reutilizar estilos em vários elementos  
* No CSS:
  * `#` representa `id`
  * `.` representa `class`  
* O `id` tem prioridade maior que a classe em caso de conflito  

### 🎨 Estilização e organização

* Seletores ajudam a deixar o código mais limpo e organizado  
* Classes facilitam reaproveitamento de estilos  
* É possível combinar mais de uma classe em um mesmo elemento  
* O uso de `span` permite destacar apenas partes específicas do texto  

## 📄 Estrutura do exercício
