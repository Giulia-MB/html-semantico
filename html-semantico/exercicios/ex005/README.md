# 📄 Exercício 005 - Módulo 01 - Capítulo 05

## 📌 Enunciado

Exercício baseado no capítulo "Caracteres, parágrafos e quebras de linha" do curso de HTML e CSS da plataforma "Curso em Vídeo". Este exercício **não foi proposto oficialmente no material do curso**, mas foi criado para complementar a seção "Inserindo Emojis"!

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/05%20-%20Caracteres%2C%20par%C3%A1grafos%20e%20quebras%20de%20linha.pdf)

## 💡 Objetivo

Praticar a inserção de emojis em páginas HTML utilizando códigos Unicode.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex005/)

## 🛠️ O que foi praticado

* Inserção de emojis em HTML  
* Uso de códigos Unicode (`&#x...;`)  
* Uso de entidades HTML  
* Estrutura básica de texto com `<p>`

### 🔍 Como encontrar emojis

Para encontrar emojis e seus códigos Unicode, foi utilizado o site:

👉 [Emojipedia](https://emojipedia.org)

## 📚 Aprendizados

* Emojis podem ser inseridos usando códigos Unicode  
* O formato utilizado em HTML é `&#xCÓDIGO;`  
* Importância do charset UTF-8 para exibição correta  
* Uso de sites como Emojipedia para encontrar códigos  

## 🚀 Resultado

Página HTML exibindo emojis corretamente no navegador.
