# 📄 Exercício 008 - Módulo 01 - Capítulo 08

## 📌 Enunciado

Exercício proposto no capítulo **"Formatação de Textos"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/08%20-%20Formata%C3%A7%C3%A3o%20de%20Textos.pdf)

## 💡 Objetivo

Explorar outras formatações de texto em HTML, incluindo exibição de código, citações, abreviações e inversão de texto.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex008/index.html)

## 🛠️ O que foi praticado

* Uso da tag `<code>` para exibição de código-fonte  
* Uso da tag `<pre>` para manter formatação original do texto  
* Uso da tag `<q>` para citações simples  
* Uso da tag `<blockquote>` para citações completas  
* Uso do atributo `cite` para referenciar a fonte da citação  
* Uso da tag `<abbr>` para abreviações  
* Uso da tag `<bdo>` para inverter a direção do texto  

## 📚 Aprendizados

* A tag `<code>` exibe texto em fonte monoespaçada, ideal para códigos  
* A tag `<pre>` preserva espaços e quebras de linha  
* Citações curtas devem usar `<q>`, enquanto citações longas usam `<blockquote>`
* O atributo `cite` pode indicar a fonte de uma citação  
* A tag `<abbr>` permite mostrar o significado de siglas ao passar o mouse  
* A tag `<bdo>` altera a direção do texto (`rtl` ou `ltr`)  
* HTML5 prioriza semântica, tornando o conteúdo mais compreensível para navegadores e mecanismos de busca  

## 🚀 Resultado

Página HTML demonstrando diferentes tipos de formatação de texto, incluindo código, citações, abreviações e manipulação de direção de texto.
