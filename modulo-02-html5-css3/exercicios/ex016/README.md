## 🛠️ O que foi praticado

### 🎨 cor01.html — Representação de cores

* Uso do atributo `style` para aplicar cores diretamente nos elementos  
* Definição de cor pelo nome (`orange`, `brown`)  
* Definição de cor por código hexadecimal (`#ffa500`, `#A52A2A`)  
* Definição de cor por `rgb()`  
* Definição de cor por `hsl()`  
* Uso de transparência com `hsla()`  
* Uso de transparência com `rgba()`  

### 🌈 cor02.html — Gradientes

* Criação de fundo com `linear-gradient()`  
* Uso de múltiplas tonalidades em sequência  
* Aplicação do atributo `background-attachment: fixed`  
* Estudo de direção do gradiente (`to right`)  
* Introdução ao conceito de `radial-gradient()`  

### 🖥️ cor03.html — Site de exemplo com CSS externo

* Criação de uma página de exemplo utilizando CSS externo  
* Aplicação de gradiente no fundo da página  
* Uso de `main` para centralizar o conteúdo  
* Estilização de títulos e parágrafos com cores harmônicas  
* Uso de sombra (`box-shadow`) e bordas arredondadas (`border-radius`)  

## 📚 Aprendizados

### 🎨 Representação de cores

* CSS permite representar a mesma cor de diferentes maneiras  
* Nomes de cores são simples, mas limitados  
* Hexadecimal é amplamente usado no desenvolvimento web  
* `rgb()` define intensidade de vermelho, verde e azul  
* `hsl()` trabalha com matiz, saturação e luminosidade  
* `hsla()` e `rgba()` adicionam canal alfa, permitindo controlar transparência  

### 🌈 Gradientes e harmonia visual

* Gradientes criam transições suaves entre cores  
* `linear-gradient()` permite definir direção e sequência de tons  
* `radial-gradient()` cria efeitos circulares  
* Fundos fixos deixam a experiência visual mais agradável  
* A escolha de paletas influencia diretamente na estética e usabilidade da página  

### 🎨 Ferramentas utilizadas para criar paletas

Durante o exercício, foram utilizados sites especializados para testar combinações de cores e montar paletas harmônicas:

* [Adobe Color](https://color.adobe.com/)
* [Paletton](https://paletton.com/)
* [Coolors](https://coolors.co/)

Essas ferramentas ajudam a:

* encontrar cores complementares e análogas  
* testar contraste  
* criar paletas equilibradas para interfaces
