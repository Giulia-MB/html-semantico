# 📄 Exercício 018 - Módulo 02 - Capítulo 14

## 📌 Enunciado

Exercício proposto no capítulo **"Vamos falar das Fontes"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/14%20-%20Fontes.pdf)

## 💡 Objetivo

Aprender duas formas de utilizar fontes personalizadas em páginas web: importando fontes online com Google Fonts e incorporando arquivos de fontes externas no próprio projeto.

## 🌐 Visualização

👉 [fonte01.html - Google Fonts](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/modulo-02-css/exercicios/ex018/fonte01.html)  
👉 [fonte02.html - Fontes Externas](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/modulo-02-css/exercicios/ex018/fonte02.html)

## 🛠️ O que foi praticado

### 🌐 fonte01.html — Google Fonts

* Importação de fontes online com `@import`  
* Uso de múltiplas famílias tipográficas  
* Aplicação de fontes diferentes em títulos e subtítulos  
* Uso de fontes de fallback  

### 💾 fonte02.html — Fontes externas

* Uso da regra `@font-face` para importar fonte local  
* Configuração de arquivos `.otf` e `.ttf`  
* Definição de nome personalizado para a fonte  
* Aplicação da fonte importada no título  

## 📚 Aprendizados

### 🌐 Google Fonts

* Permite importar fontes diretamente da web  
* Facilita a personalização visual sem precisar baixar arquivos  
* É ideal para fontes amplamente usadas e de fácil acesso  

### 💾 Fontes externas

* A regra `@font-face` permite usar arquivos de fonte no próprio projeto  
* Dá mais liberdade para usar fontes exclusivas  
* Funciona mesmo sem depender de conexão com serviços externos  

### 🔤 Tipografia

* A escolha da fonte influencia estética, legibilidade e identidade visual  
* É importante definir fontes de fallback para evitar problemas de carregamento
