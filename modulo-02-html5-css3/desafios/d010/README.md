# 📄 Desafio 010 - Módulo 02 - Capítulo 17

## 📌 Enunciado

Desafio proposto no material **"Projeto Android"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material do desafio](https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-02/d010/desafio-android.pdf)  
👉 [Material do capítulo 17](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/17%20-%20Primeiro%20mini-projeto.pdf)

## 💡 Objetivo

Criar um mini-projeto em formato de página de notícia, aplicando os conceitos estudados no **Capítulo 17**, como planejamento com wireframe, responsividade, variáveis em CSS, seletor global, sombras em textos, personalização de listas e incorporação flexível de vídeos.

## 🌐 Visualização

👉 [Projeto Android](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/modulo-02-css/desafios/d010/android.html)

## 🛠️ O que foi praticado

* Planejamento visual do projeto a partir de um **wireframe**  
* Construção de página com estrutura semântica usando `header`, `nav`, `main`, `article`, `aside` e `footer`  
* Uso de **responsividade** com `min-width` e `max-width`  
* Criação de variáveis CSS em `:root` para cores e fontes  
* Uso do seletor global `*` para resetar margens e paddings  
* Aplicação de `line-height` para melhorar leitura dos textos  
* Uso de `text-shadow` para destacar textos do cabeçalho  
* Personalização de listas com `list-style-type`, `columns` e `list-style-position`  
* Inserção de imagens responsivas com `<picture>`  
* Incorporação de vídeo do YouTube com `iframe` dentro de contêiner flexível usando `position: relative` e `position: absolute`  
* Uso de fontes externas com Google Fonts e `@font-face`  

## 📚 Aprendizados

* Antes de começar a programar, é importante planejar a estrutura da página com um **wireframe**  
* Um site **responsivo** se adapta melhor a telas de tamanhos diferentes  
* Variáveis CSS facilitam manutenção e padronização do projeto  
* O seletor `*` ajuda a remover estilos automáticos do navegador  
* A propriedade `line-height` melhora bastante a leitura de textos longos  
* `text-shadow` cria contraste e destaque visual em títulos  
* Listas podem ser mais organizadas e bonitas com colunas e marcadores personalizados  
* Vídeos incorporados podem ser tornados mais flexíveis usando contêiner e posicionamento em CSS  
* Esse desafio une vários conteúdos do módulo em um único projeto prático  

## 📄 Estrutura do desafio

O desafio contém:
