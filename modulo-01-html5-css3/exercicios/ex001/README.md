# 📄 Exercício 001 - Módulo 1 - Capítulo 4

## 📌 Enunciado

Exercício proposto no material **"Primeiros passos em HTML"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/04%20-%20Primeiros%20passos%20HTML.pdf)

## 💡 Objetivo

Criar o primeiro documento HTML, utilizando a estrutura básica de uma página web.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex001/)

## 🛠️ O que foi praticado

* Estrutura básica do HTML (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
* Uso de título (`<title>`)
* Inserção de conteúdo com `<h1>` e `<p>`
* Criação de arquivo `index.html`

## 📚 Aprendizados

* Como funciona a base de uma página web
* Organização inicial de código
* Uso do VS Code para gerar estrutura automática (`! + Enter`)

## 🚀 Resultado

Primeira página HTML criada com sucesso.
