# 📄 Exercício 009 - Módulo 01 - Capítulo 09

## 📌 Enunciado

Exercício proposto no capítulo **"Listas com HTML5"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/09%20-%20Listas%20HTML.pdf)

### 📝 Observação

O exercício original foi adaptado:  
em vez de replicar exatamente o exemplo do vídeo, foi utilizado o **exercício proposto no material**, criando uma lista personalizada com jogos favoritos organizados por plataforma.

## 💡 Objetivo

Praticar a criação e organização de diferentes tipos de listas em HTML, incluindo listas ordenadas, não ordenadas, listas aninhadas e listas de definição.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex009/index.html)

## 🛠️ O que foi praticado

* Uso da tag `<ol>` para listas ordenadas  
* Uso da tag `<ul>` para listas não ordenadas  
* Uso da tag `<li>` para itens de lista  
* Uso de atributos como `type` e `start` para personalização  
* Criação de listas aninhadas (listas dentro de listas)  
* Uso da tag `<dl>` para listas de definição  
* Uso das tags `<dt>` e `<dd>` para termos e descrições  

## 📚 Aprendizados

* Listas ajudam a organizar e facilitar a leitura do conteúdo :contentReference[oaicite:0]{index=0}  
* Listas ordenadas (`<ol>`) são usadas quando a ordem dos itens é importante  
* Listas não ordenadas (`<ul>`) são usadas quando a ordem não importa  
* É possível misturar diferentes tipos de listas (listas aninhadas) :contentReference[oaicite:1]{index=1}  
* O atributo `type` altera o estilo da lista (números, letras, romanos, etc.) :contentReference[oaicite:2]{index=2}  
* O atributo `start` define o ponto inicial da contagem  
* Listas de definição (`<dl>`) relacionam termos (`<dt>`) com descrições (`<dd>`) :contentReference[oaicite:3]{index=3}
