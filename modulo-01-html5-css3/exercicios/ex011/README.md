# 📄 Exercício 011 - Módulo 01 - Capítulo 11

## 📌 Enunciado

Exercício proposto no capítulo **"Imagens dinâmicas, áudios e vídeos"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/11%20-%20Imagens%20din%C3%A2micas%2C%20%C3%A1udios%20e%20v%C3%ADdeos.pdf)

## 💡 Objetivo

Praticar o uso de mídias em HTML5, incluindo imagens responsivas e reprodução de áudio no navegador.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex011/index.html)

## 🛠️ O que foi praticado

* Uso da tag `<picture>` para imagens dinâmicas  
* Uso da tag `<source>` com media queries para imagens responsivas  
* Adaptação de imagens conforme o tamanho da tela  
* Redimensionamento de imagens no GIMP para diferentes dispositivos  
* Uso da tag `<audio>` para reprodução de som  
* Uso de múltiplos formatos de áudio (`.mp3`, `.ogg`, `.wav`)  
* Uso dos atributos `controls`, `autoplay`, `loop` e `preload`  
* Criação de link alternativo para download do áudio  

## 📚 Aprendizados

* A tag `<picture>` permite exibir imagens diferentes conforme a largura da tela  
* A ordem das tags `<source>` é importante para que o navegador escolha corretamente a imagem ideal  
* O atributo `media` permite definir condições de largura para imagens responsivas  
* As imagens foram geradas no software GIMP em três tamanhos:
  - Pequena: **300x300 px**
  - Média: **700x700 px**
  - Grande: **1000x1000 px**
* A tag `<audio>` facilita a reprodução de mídias diretamente no navegador  
* O atributo `controls` exibe os controles de reprodução  
* O atributo `preload="metadata"` carrega apenas informações básicas do áudio  
* O uso de múltiplos formatos aumenta a compatibilidade entre navegadores  
* É importante fornecer uma alternativa de download caso o navegador não consiga reproduzir o arquivo  

## 📄 Estrutura do exercício

O exercício apresenta:

* seção de imagem dinâmica com adaptação para diferentes tamanhos de tela  
* imagens redimensionadas no GIMP para testes de responsividade  
* seção de reprodução de áudio com controles e repetição automática  
* fallback com link para download do arquivo de áudio  

## 🚀 Resultado

Página HTML com imagem responsiva e reprodução de áudio, demonstrando o uso de recursos multimídia no HTML5.
