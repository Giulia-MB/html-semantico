# 📄 Exercício 007 - Módulo 01 - Capítulo 08

## 📌 Enunciado

Exercício proposto no capítulo **"Formatação de Textos"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/08%20-%20Formata%C3%A7%C3%A3o%20de%20Textos.pdf)

## 💡 Objetivo

Explorar as principais formatações de texto em HTML e compreender a diferença entre marcações antigas focadas em aparência e marcações semânticas utilizadas no HTML5.

## 🌐 Visualização

👉 [html4.html](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex007/html4.html)  
👉 [html5.html](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex007/html5.html)  
👉 [index.html](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex007/index.html)

## 🛠️ O que foi praticado

* Comparação entre HTML4 e HTML5  
* Uso de marcações semânticas como `<strong>`, `<em>` e `<address>`  
* Uso de formatações de texto como `<mark>`, `<small>`, `<del>`, `<ins>`, `<sup>` e `<sub>`  
* Entendimento sobre tags obsoletas e substituições mais adequadas  

## 📚 Aprendizados

* O HTML5 prioriza **semântica**, enquanto a aparência visual deve ser controlada com CSS  
* Tags antigas como `<center>`, `<font>`, `<marquee>` e o atributo `bgcolor` são exemplos de recursos obsoletos  
* Em vez de apenas sublinhar um endereço, faz mais sentido usar a tag `<address>`  
* Para negrito e itálico com significado, é preferível usar `<strong>` e `<em>`  
* Algumas tags antigas ainda podem funcionar no navegador, mas não são recomendadas em projetos atuais  
* Textos sobrescritos e subscritos podem ser criados com `<sup>` e `<sub>`  

## 📄 Estrutura do exercício

Este exercício foi dividido em três arquivos:

* `html4.html` — exemplo de código antigo, com recursos obsoletos  
* `html5.html` — exemplo com foco em semântica no HTML5  
* `index.html` — demonstração das principais formatações de texto estudadas

## 🚀 Resultado

Conjunto de páginas HTML demonstrando a evolução do HTML4 para o HTML5 e o uso correto das principais formatações de texto.
