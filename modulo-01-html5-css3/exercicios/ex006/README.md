# 📄 Exercício 006 - Módulo 01 - Capítulo 07

## 📌 Enunciado

Exercício proposto no capítulo **"Hierarquia de Títulos"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/07%20-%20Hierarquia%20de%20t%C3%ADtulos.pdf)

## 💡 Objetivo

Criar uma página HTML organizada utilizando corretamente a hierarquia de títulos (`<h1>` até `<h4>`), representando a estrutura de um texto com temas e subtemas.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex006/)

## 🛠️ O que foi praticado

* Uso das tags de título `<h1>` até `<h4>`  
* Organização hierárquica do conteúdo  
* Uso de parágrafos `<p>` para desenvolvimento dos tópicos  
* Estruturação semântica de uma página HTML  

## 📚 Aprendizados

* As tags `<h1>` até `<h6>` representam níveis de título em HTML  
* `<h1>` é o título principal e define o tema central da página  
* `<h2>`, `<h3>` e demais níveis representam subtópicos em ordem hierárquica  
* A hierarquia deve ser respeitada (não pular níveis)  
* O uso correto melhora a organização, leitura e acessibilidade do conteúdo :contentReference[oaicite:0]{index=0}  

## 🚀 Resultado

Página HTML estruturada com títulos e subtítulos organizados corretamente, representando a hierarquia de um conteúdo textual.
