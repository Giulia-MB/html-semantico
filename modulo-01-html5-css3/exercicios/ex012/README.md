# 📄 Exercício 012 - Módulo 01 - Capítulo 11

## 📌 Enunciado

Exercício proposto no capítulo **"Imagens dinâmicas, áudios e vídeos"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/11%20-%20Imagens%20din%C3%A2micas%2C%20%C3%A1udios%20e%20v%C3%ADdeos.pdf)

## 💡 Objetivo

Praticar a inserção de vídeos em páginas HTML5, utilizando arquivos hospedados localmente e vídeos incorporados de plataformas externas.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex012/index.html)

## 🛠️ O que foi praticado

* Uso da tag `<video>` para vídeos hospedados localmente  
* Uso de múltiplos formatos de vídeo para compatibilidade entre navegadores  
* Uso dos atributos `controls`, `autoplay`, `loop` e `poster`  
* Uso da tag `<iframe>` para incorporar vídeos externos  
* Incorporação de vídeos do YouTube e Vimeo  
* Conversão de vídeos para diferentes formatos  

## 📚 Aprendizados

* A tag `<video>` permite reproduzir vídeos diretamente no navegador  
* O atributo `poster` exibe uma imagem de capa antes da reprodução  
* O uso de múltiplos formatos aumenta a compatibilidade entre navegadores  
* O atributo `controls` adiciona controles de reprodução ao vídeo  
* O atributo `autoplay` inicia automaticamente (dependendo das permissões do navegador)  
* A tag `<iframe>` permite incorporar vídeos de plataformas externas  

### 🎞️ Conversão dos vídeos

Para este exercício:

* O vídeo original foi baixado no site **Pexels**  
* O arquivo original em **MP4** foi mantido  
* Os formatos **WEBM** e **MKV** foram convertidos no software **HandBrake**  
* O formato **OGV** foi convertido utilizando uma ferramenta online  

### 📌 Formatos utilizados

* `meu-video.mp4` → vídeo original  
* `meu-video.webm` → convertido no HandBrake  
* `meu-video.mkv` → convertido no HandBrake  
* `meu-video.ogv` → convertido online  

## 📄 Estrutura do exercício

O exercício apresenta:

* vídeo hospedado localmente com formato MP4  
* imagem de capa (`poster`) para o vídeo local  
* vídeo incorporado do YouTube  
* vídeo incorporado do Vimeo  

## 🚀 Resultado

Página HTML com reprodução de vídeos locais e vídeos incorporados de plataformas externas, demonstrando diferentes formas de exibir mídia em HTML5.

---

## 💬 Observação

Devido ao limite de tamanho de arquivos suportado pelo GitHub para uploads em repositórios, não foi possível manter todos os formatos de vídeo convertidos no projeto publicado.  

Por isso, a versão final disponível no repositório contém apenas os arquivos essenciais para demonstrar o funcionamento do exercício.  

Além disso, foi identificado um erro **153** na reprodução do vídeo incorporado do YouTube. Esse problema está relacionado a restrições da própria plataforma ou políticas de reprodução do navegador, e não à estrutura do código HTML.
