# 📄 Exercício 002 - Módulo 1 - Capítulo 5

## 📌 Enunciado

Exercício baseado no capítulo **"Caracteres, parágrafos e quebras de linha"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/05%20-%20Caracteres%2C%20par%C3%A1grafos%20e%20quebras%20de%20linha.pdf)

## 💡 Objetivo

Compreender como o HTML interpreta textos, parágrafos e quebras de linha, além de aprender a utilizar caracteres especiais corretamente.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex002/)

## 🛠️ O que foi praticado

* Uso da tag `<p>` para criação de parágrafos
* Uso da tag `<br>` para quebra de linha
* Diferença entre pressionar Enter e usar HTML corretamente
* Utilização de **entidades HTML** (como `&lt;` e `&gt;`)
* Inserção de símbolos e emojis

## 📚 Aprendizados

* O navegador **ignora quebras de linha feitas com Enter** no código
* Apenas as **tags HTML controlam o layout do texto**
* A tag `<br>` é responsável por quebrar linha
* Não é recomendado usar vários `<br>` para criar espaçamento (isso deve ser feito com CSS)
* Para exibir símbolos especiais (como `<` e `>`), usamos entidades HTML

## 🚀 Resultado

Página HTML contendo parágrafos, quebras de linha e uso de caracteres especiais.
