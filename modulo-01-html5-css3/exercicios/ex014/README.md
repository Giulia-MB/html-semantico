# 📄 Exercício 014 - Módulo 01 - Capítulo 12

## 📌 Enunciado

Exercício proposto no capítulo **"Trabalhando com estilos"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/12%20-%20Trabalhando%20com%20estilos.pdf)

## 💡 Objetivo

Praticar a aplicação de estilos CSS locais / internos, separando a estrutura HTML da estilização dentro do mesmo arquivo.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex014/index.html)

## 🛠️ O que foi praticado

* Uso da tag `<style>` dentro do `<head>`  
* Separação entre estrutura HTML e estilo visual  
* Definição de cor de fundo da página  
* Configuração de fonte padrão e tamanho base  
* Estilização global de títulos `<h1>` e `<h2>`  
* Alinhamento justificado dos parágrafos  

## 📚 Aprendizados

* O CSS interno permite centralizar os estilos em um único bloco dentro do arquivo HTML  
* Essa abordagem deixa o código mais organizado do que o CSS inline  
* É possível aplicar estilos em vários elementos de uma só vez  
* Facilita manutenção em páginas simples ou testes rápidos  
* O uso de seletores como `body`, `h1`, `h2` e `p` torna a estilização mais prática

### ⚠️ Desvantagens do CSS interno

* O bloco de estilos pode ficar maior que o próprio conteúdo da página  
* O CSS continua preso ao arquivo HTML  
* Se houver mais páginas, é necessário copiar o mesmo estilo em cada uma  
* Não é a melhor solução para projetos maiores  

## 🚀 Resultado

Página HTML estilizada com CSS interno, mostrando uma forma mais organizada de aplicar estilos em comparação ao CSS inline.

---

## 💬 Observação

Este exercício mostra uma evolução importante no uso do CSS, ao separar melhor o conteúdo da apresentação visual. Ainda assim, também evidencia as limitações de manter os estilos presos ao mesmo arquivo HTML.
