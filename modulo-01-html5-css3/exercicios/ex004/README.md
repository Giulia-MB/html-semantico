# 📄 Exercício 004 - Módulo 01 - Capítulo 06

## 📌 Enunciado

Exercício proposto no capítulo **"Imagens e Favicon"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/06%20-%20Imagens%20e%20Favicon.pdf)

## 💡 Objetivo

Aprender a adicionar um favicon em uma página HTML e conhecer diferentes formas de criar esse ícone.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex004/)

## 🛠️ O que foi praticado

* Inserção de favicon com a tag `<link>` dentro do `<head>`  
* Uso de arquivos `.ico`  
* Organização de arquivos do projeto  

## 🎨 Formas de criar um favicon

Durante a aula, foram apresentados diferentes métodos para criar um favicon:

### 🔹 1. Baixar ícones prontos
- Utilizando sites como o Icon Archive  

### 🔹 2. Criar manualmente
- Usando o site Favicon.cc  
- Permite desenhar o ícone pixel por pixel  

### 🔹 3. Gerar a partir de imagem
- Utilizando o site Favicon.io  
- Converte imagens para favicon automaticamente  

### 🔹 4. Gerar a partir de emoji
- Também no Favicon.io  
- Permite criar favicon usando emojis  

## 📚 Aprendizados

* O favicon é o ícone exibido na aba do navegador  
* Ajuda na identificação visual do site  
* Pode ser criado de diferentes formas  
* Deve ser adicionado dentro da tag `<head>`  

## 🚀 Resultado

Página HTML com favicon exibido corretamente na aba do navegador.
