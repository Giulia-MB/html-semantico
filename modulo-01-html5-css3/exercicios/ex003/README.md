# 📄 Exercício 003 - Módulo 1 - Capítulo 6

## 📌 Enunciado

Exercício proposto no capítulo **"Imagens e Favicon"** do curso de HTML e CSS da plataforma "Curso em Vídeo". 

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/06%20-%20Imagens%20e%20Favicon.pdf)

## 💡 Objetivo

Praticar a inserção de imagens em uma página HTML utilizando a tag apropriada.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex003/)

## 🛠️ O que foi praticado

* Uso da tag `<img>` para inserir imagens  
* Configuração de caminhos de arquivos (relativos)  
* Uso de atributos como `src` e `alt`  
* Organização de imagens dentro do projeto  

## 📚 Aprendizados

* Como adicionar imagens em páginas web  
* Importância do atributo `alt` para acessibilidade  
* Diferença entre caminhos relativos e absolutos  
* Boas práticas na organização de arquivos 

## 🚀 Resultado

Página HTML exibindo imagens corretamente.
