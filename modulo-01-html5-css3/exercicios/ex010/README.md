# 📄 Exercício 010 - Módulo 01 - Capítulo 10

## 📌 Enunciado

Exercício proposto no capítulo **"Ligações em toda parte"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/10%20-%20Liga%C3%A7%C3%B5es%20em%20toda%20parte.pdf)

## 💡 Objetivo

Praticar a criação de hyperlinks em HTML, incluindo links externos, links internos entre páginas e links para download de arquivos.

## 🌐 Visualização

👉 [index.html](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex010/index.html)  

## 🛠️ O que foi praticado

* Uso da tag `<a>` para criação de links  
* Uso do atributo `href` para definir o destino do link  
* Uso do atributo `target` (`_blank` e `_self`)  
* Uso do atributo `rel` (`external`, `next`, `prev`, `nofollow`)  
* Criação de links externos (para outros sites)  
* Criação de links internos (entre páginas do próprio site)  
* Navegação entre páginas (`index.html` e `pag002.html`)  
* Criação de links para download de arquivos (`download` e `type`)  

## 📚 Aprendizados

* Links são a base da navegação na web, permitindo conectar páginas e conteúdos
* A tag `<a>` é utilizada para criar hyperlinks  
* O atributo `href` define o destino do link  
* O atributo `target="_blank"` abre o link em uma nova aba, enquanto `_self` mantém na mesma página 
* O atributo `rel` indica a relação do link (ex: `external`, `next`, `prev`, `nofollow`)  
* Links internos não precisam de URL completa, apenas o caminho do arquivo 
* Para downloads, usamos `download` e `type` para indicar o tipo do arquivo (ex: `application/pdf`) 

## 📄 Estrutura do exercício

O exercício contém:

* `index.html`  
  - Links externos para GitHub e YouTube  
  - Links internos para outras páginas do site  
  - Links para download de arquivos (PDF e ZIP) 

## 🚀 Resultado

Pequeno site com múltiplas páginas interligadas, demonstrando navegação interna, links externos e download de arquivos.

## 💬 Observação

Este exercício é fundamental para entender a base da navegação na web, já que os hyperlinks são responsáveis por conectar conteúdos e permitir a experiência de navegação entre páginas.
