# 📄 Exercício 013 - Módulo 01 - Capítulo 12

## 📌 Enunciado

Exercício proposto no capítulo **"Trabalhando com estilos"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/aulas-pdf/12%20-%20Trabalhando%20com%20estilos.pdf)

## 💡 Objetivo

Praticar a aplicação de estilos CSS inline diretamente nas tags HTML, entendendo como personalizar cores, fontes, tamanhos e alinhamento de textos.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/exercicios/ex013/index.html)

## 🛠️ O que foi praticado

* Aplicação de CSS inline com o atributo `style`  
* Alteração da cor de fundo da página  
* Definição de fonte e tamanho base do texto  
* Estilização de títulos `<h1>` e `<h2>`  
* Alinhamento de parágrafos com `text-align: justify`  
* Controle de tamanhos com `em`  

## 📚 Aprendizados

* O CSS inline permite aplicar estilos diretamente em cada elemento HTML  
* É possível controlar aparência de texto, cores e espaçamento sem arquivo externo  
* O uso de `font-family`, `font-size`, `color` e `background-color` ajuda a personalizar a interface  
* O alinhamento justificado melhora a organização visual dos parágrafos

### ⚠️ Desvantagens do CSS inline

* Polui o código HTML, misturando estrutura com estilo  
* Dificulta manutenção e leitura do código  
* Obriga repetição de estilos em várias tags  
* Não é recomendado para projetos maiores  

## 🚀 Resultado

Página HTML estilizada com CSS inline, demonstrando como aplicar estilos diretamente nos elementos da página.

---

## 💬 Observação

Este exercício é importante para compreender os primeiros conceitos de CSS, mas também mostra por que essa abordagem não é a mais indicada em projetos reais, já que dificulta manutenção e reaproveitamento de estilos.
