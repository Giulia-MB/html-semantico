# 📄 Desafio 002 - Módulo 01 - Capítulo 06

## 📌 Enunciado

Desafio proposto no material **"Desafio das Imagens"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-01/d002/desafio-imagens.pdf)

## 💡 Objetivo

Criar uma página HTML que contenha:
- Uma imagem no conteúdo da página  
- Um favicon (ícone de favoritos)

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/desafios/d002/)

## 🛠️ O que foi praticado

* Uso da tag `<img>` para inserir imagem no conteúdo  
* Inserção de favicon com a tag `<link>` no `<head>`  
* Uso de diferentes formatos de imagem  
* Escolha adequada de arquivos conforme tamanho e finalidade  
* Organização de arquivos dentro do projeto  

## 📚 Aprendizados

* Diferença entre imagem de conteúdo e favicon  
* Importância de escolher corretamente o tamanho e formato das imagens  
* Uso de caminhos relativos para acessar arquivos  
* Organização de recursos (imagens e ícones) em projetos web  

## 🚀 Resultado

Página HTML exibindo uma imagem no conteúdo e um favicon na aba do navegador, conforme o desafio proposto.
