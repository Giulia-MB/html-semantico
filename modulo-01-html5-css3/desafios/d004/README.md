# 📄 Desafio 004 - Módulo 01 - Capítulos 05 e 09

## 📌 Enunciado

Desafio proposto no material **"Desafio dos Emojis"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-01/d004/desafio-emoji.pdf)

### 📝 Proposta do desafio

Criar uma página HTML contendo uma lista de emojis organizados em 6 categorias:

- Carinhas  
- Mãos  
- Animais  
- Esportes  
- Comidas  
- Objetos  

Cada categoria deve conter **3 emojis diferentes**, acompanhados de seus respectivos códigos Unicode.

⚠️ Importante:
- Não utilizar os mesmos emojis do exemplo do professor  
- Buscar os códigos em fontes confiáveis  

## 💡 Objetivo

Praticar a criação de listas em HTML e a inserção de emojis utilizando códigos Unicode.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/desafios/d004/index.html)

## 🛠️ O que foi praticado

* Uso da tag `<ul>` para listas não ordenadas  
* Uso da tag `<li>` para itens de lista  
* Organização de conteúdo em categorias  
* Inserção de emojis utilizando códigos Unicode (`&#x...;`)  
* Estruturação de conteúdo com títulos (`<h1>` e `<h2>`)  

## 📚 Aprendizados

* Emojis podem ser inseridos em HTML utilizando códigos Unicode  
* O formato correto em HTML é:

  ```html
  &#xCÓDIGO;
  ```
* Os códigos podem ser encontrados em sites como:

  👉 [Emojipedia](https://emojipedia.org/)

* Listas ajudam a organizar informações de forma clara
* A combinação de listas + emojis melhora a apresentação visual do conteúdo
  
### 🔍 Como usar o Unicode

1. Acesse o site [Emojipedia](https://emojipedia.org)  
2. Escolha um emoji  
3. Copie o código (ex: `U+1F60E`)  
4. No HTML, utilize:

```html
&#x1F60E;
```
### 👉 Passos:

* Remover o U+
* Adicionar &#x no início
* Adicionar ; no final
  
## 🚀 Resultado

Página HTML exibindo uma lista organizada de emojis em diferentes categorias, utilizando códigos Unicode personalizados.
