# 📄 Desafio 008 - Módulo 01 - Capítulo 12

## 📌 Enunciado

Desafio proposto no material **"Criando uma Navegação"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-01/d008/desafio-navegacao.pdf)

## 💡 Objetivo

Praticar a navegação entre páginas HTML utilizando links internos.

## 🌐 Visualização

👉 [Página principal](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/desafios/d008/index.html)  
👉 [Página verde](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/desafios/d008/pag-verde.html)  
👉 [Página amarela](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/desafios/d008/pag-amarela.html)

## 🛠️ O que foi praticado

* Criação de links internos com a tag `<a>`  
* Navegação entre páginas do mesmo projeto  
* Organização de arquivos HTML em um mesmo diretório  
* Aplicação de estilos simples para diferenciar páginas  

## 📚 Aprendizados

* A navegação entre páginas é feita com links internos usando caminhos relativos  
* É importante permitir que o usuário volte facilmente à página inicial  
* A experiência de navegação melhora quando as páginas têm identidade visual clara  
* Esse desafio reforça a base da estrutura de pequenos sites com múltiplas páginas :contentReference[oaicite:0]{index=0}  

## 📄 Estrutura do desafio

O desafio contém:

* página principal cinza com links para outras áreas  
* página verde com link de retorno  
* página amarela com link de retorno  

## 🚀 Resultado

Pequeno site com navegação entre páginas, permitindo acessar áreas diferentes e retornar ao índice.

---

## 💬 Observação

Este desafio é uma ótima introdução ao conceito de navegação entre páginas, que é a base para a construção de sites completos com múltiplas seções.
