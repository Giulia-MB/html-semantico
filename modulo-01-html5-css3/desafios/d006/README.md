# 📄 Desafio 006 - Módulo 01 - Capítulo 10

## 📌 Enunciado

Desafio proposto no material **"Desafio das Tags"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-01/d006/desafio-tags.pdf)

### 📝 Proposta do desafio

Criar uma página HTML com uma lista de elementos que podem ser adicionados em uma página e, ao passar o mouse sobre cada termo, exibir qual tag HTML deve ser utilizada para construí-lo.

O desafio sugere pelo menos 7 itens na lista, mas é possível adicionar mais. 

## 💡 Objetivo

Praticar a criação de listas em HTML e o uso de dicas visuais com o atributo `title`, permitindo exibir a tag correspondente ao passar o mouse sobre cada item.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/desafios/d006/index.html)

## 🛠️ O que foi praticado

* Uso de listas não ordenadas com `<ul>` e `<li>`  
* Uso da tag `<abbr>` para exibir dicas ao passar o mouse  
* Uso do atributo `title` para mostrar a tag HTML correspondente  
* Estruturação de conteúdo com `<h1>`, `<p>` e listas  
* Revisão prática de várias tags HTML estudadas até aqui  

## 📚 Aprendizados

* O atributo `title` pode ser usado para exibir informações adicionais ao passar o mouse  
* A tag `<abbr>` ajuda a associar um texto visível a uma explicação complementar  
* Listas ajudam a organizar conteúdos de forma clara e objetiva  
* Esse desafio funciona como revisão prática de várias tags HTML estudadas anteriormente  

## 🚀 Resultado

Página HTML com uma lista de elementos e dicas interativas, mostrando ao usuário qual tag HTML deve ser usada em cada caso.

## 💬 Observação

Este desafio é uma revisão prática de vários conteúdos já estudados no módulo, especialmente estruturação de texto, listas e uso de atributos em HTML.
