# 📄 Desafio 007 - Módulo 01 - Capítulo 11

## 📌 Enunciado

Desafio proposto no material **"Imagem flexível"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-01/d007/desafio-imagem-flexivel.pdf)

### 📝 Proposta do desafio

Criar uma página HTML que exiba uma imagem responsiva, alterando automaticamente o arquivo carregado de acordo com a largura da janela do navegador.

## 💡 Objetivo

Praticar o uso de imagens flexíveis em HTML5, adaptando o conteúdo visual para diferentes tamanhos de tela sem usar JavaScript.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/desafios/d007/index.html)

## 🛠️ O que foi praticado

* Uso da tag `<picture>` para imagens responsivas  
* Uso da tag `<source>` com media queries  
* Adaptação de imagens conforme a largura da tela  
* Organização de arquivos em diferentes tamanhos  
* Estruturação simples de conteúdo com `<h1>` e `<p>`  

## 📚 Aprendizados

* A tag `<picture>` permite trocar imagens de acordo com o tamanho da tela  
* A ordem das tags `<source>` é importante para o funcionamento correto  
* O atributo `media` define em que situação cada imagem será usada  
* Não é necessário JavaScript para criar esse tipo de adaptação  
* O uso de imagens adequadas melhora a experiência do usuário em diferentes dispositivos

## 📄 Estrutura do desafio

O desafio apresenta:

* um título principal  
* um pequeno texto explicativo  
* uma imagem flexível que muda conforme o tamanho da janela  

## 🚀 Resultado

Página HTML com imagem responsiva, carregando versões diferentes da mesma paisagem de acordo com a largura do navegador.

---

## 💬 Observação

O material do desafio reforça a importância de utilizar imagens livres de direitos autorais e sugere o uso de bancos gratuitos, como o Pexels, para esse tipo de exercício.
