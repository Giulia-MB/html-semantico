# 📄 Desafio 005 - Módulo 01 - Capítulo 10

## 📌 Enunciado

Desafio proposto no material **"Meu perfil social"** do curso de HTML e CSS da plataforma "Curso em Vídeo".

👉 [Material original](https://github.com/gustavoguanabara/html-css/blob/master/desafios/modulo-01/d005/desafio-social.pdf)

### 📝 Proposta do desafio

Criar uma página HTML com:

- uma foto de perfil  
- uma breve descrição pessoal  
- uma lista com links para redes sociais e perfis profissionais  

## 💡 Objetivo

Praticar a criação de links em HTML e desenvolver uma página simples de apresentação pessoal.

## 🌐 Visualização

👉 [Abrir exercício](https://giulia-mb.github.io/FrontEnd-NTTdata/html-semantico/desafios/d005/index.html)

## 🛠️ O que foi praticado

* Uso da tag `<a>` para links externos  
* Uso dos atributos `href`, `target` e `rel`  
* Uso da tag `<img>` para foto de perfil e ícones  
* Uso de listas com `<ul>` e `<li>`  
* Estruturação de conteúdo com títulos e parágrafos  

## 📚 Aprendizados

* Links são fundamentais para conectar páginas e perfis na web  
* O atributo `target="_blank"` abre o link em nova aba  
* O atributo `rel="external"` indica que o destino é um site externo  
* A organização em lista facilita a leitura das redes sociais  
* Uma página simples em HTML já pode funcionar como apresentação profissional inicial  

## 📄 Estrutura do desafio

A página apresenta:

* título principal  
* seção de apresentação pessoal  
* foto de perfil  
* descrição curta  
* lista com links para redes sociais e perfis profissionais  

## 🚀 Resultado

Página HTML de perfil social com foto, descrição e links para redes externas.

## 💬 Observação

O próprio desafio orienta que a página deve ser personalizada com os dados e links do aluno, sem copiar o perfil do professor. Também recomenda o uso de uma imagem de perfil em tamanho 150x150 pixels e reforça a importância de manter perfis no GitHub e no LinkedIn.
